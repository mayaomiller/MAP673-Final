### Additional Map Info

1. Identify your anticipated methods of thematic representation methods (e.g., icons, choropleth, prop symbols, etc.).
A choropleth map does not fit with identifying the locations of historic sites. However, I do think it would be valuable to have a secondary map that displays the population density in a choropleth map to show which sites (and what types of historic sites) are closer to densly populated areas versus which ones are not. The map identifying the location of sites will utilize points. Although prop symbols would be fun and exciting, I think it could potentially make the map very busy. It would be easier for the viewer to understand the map at a glance with various colored dots. 
2. Briefly describe your anticipated user interface (UI).
I think my UI will be similar to the webpages we had toward the beginning of class rather than a singular map taking up thw whole screen. I think it is valuable to have the ability to scroll down the webpage and be guided through the information and see the maps, associated information, and pictures. 

### Map objectives and user needs

- Why does the map need to be made? Consider your position to the mapping project. Why are you the one to be designing it?
This map is useful in displaying the various types of historical and cultural sites the Unites States contains. Many people don't know the variety and scale the country has to offer. I think this type of map is not only a wonderful tool to help people know what history is located near them, but also to show that there are alternative tourist locations outside of the main dozen in the United States as it relates to historical sites. 
- What type of user do you expect to use this map and what needs do you anticipate they will have?
I expect the user to be semi interested in history or culture. There aren't too many specific needs I anticipate other than a user friendly interface. Since my goals for the map are largely for an audience, the map needs to be user friendly and simple to understand.
